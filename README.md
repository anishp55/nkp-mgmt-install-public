# nkp-mgmt-install

this requires a self-hosted running with the following binaries installed
    nkp
    kubectl
    docker or podman

you will need to define the following secrets
    NUTANIX_USER
    NUTANIX_PASSWORD
    REGISTRY_USER
    REGISTRY_PASSWORD

you will need to update the nkp-vars.sh file to reflect your environment
